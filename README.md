# ARAFAT — To-Do List Web App

A sleek, fully functional **To-Do List Web Application** built with pure HTML, CSS, and JavaScript. No frameworks, no installation — just open and use.

---

## 🔴 Live Demo

> Open `todo-app.html` directly in any modern browser — no server required.

---

## ✨ Features

- ✅ **Add Tasks** — Type and press `Enter` or click `+ Add`
- ✏️ **Edit Tasks** — Inline editing with `Enter` to save, `Escape` to cancel
- 🗑️ **Delete Tasks** — Smooth slide-out animation on removal
- ☑️ **Complete Tasks** — Click checkbox to mark done with strikethrough
- 🎯 **Priority Levels** — High 🔴 / Medium 🟡 / Low 🟢
- 🔍 **Filter Tasks** — All, Active, Completed, or by Priority
- 📊 **Live Counters** — Total / Done / Pending updated in real-time
- 💾 **LocalStorage** — Tasks persist across browser sessions

## 💡 How to Use

| Action | How |
|---|---|
| Add a task | Type in the input field → Press `Enter` or click `+ Add` |
| Set priority | Select High / Med / Low from the dropdown before adding |
| Complete a task | Click the checkbox on the left |
| Edit a task | Hover over task → Click ✎ icon → Edit → Press `Enter` |
| Delete a task | Hover over task → Click ✕ icon |
| Filter tasks | Click filter buttons: All / Active / Completed / Priority |

---

## 📌 Notes

- All tasks are saved in the browser's `localStorage` under the key `arafat_tasks`
- No backend or database required
- Works offline — fully client-side
